# Chrome Extension Installation Guide

## Windows

1. Open **Google Chrome** on your Windows PC.
2. Click the three-dot menu (**⋮**) in the top-right corner of the browser.
3. Navigate to **Extensions** > **Manage Extensions** (or type `chrome://extensions` in the address bar).
4. Enable **Developer mode** using the toggle in the top-right corner of the Extensions page.
5. Click the **Load unpacked** button that appears.
6. In the file dialog, browse to the folder containing the extension files and select it.
7. The extension will appear in your extensions list and is now ready to use.
8. (Optional) Click the puzzle piece icon in the Chrome toolbar and pin the extension for quick access.

## Troubleshooting

- **Extension not loading?** Make sure you selected the folder that contains `manifest.json`, not a parent or subfolder.
- **Permission errors?** Close and reopen Chrome, then try loading the extension again.
- **Extension disappeared after restart?** Ensure Developer mode is still enabled on `chrome://extensions`.
